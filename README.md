Вашему коллеге-программисту досталась задача о сортировке фруктов.
Он начал ее делать, но неожиданно ушел на больничный.
«Подхватить» и завершить задачу поручили вам.

### Техническое задание
В деревянном, металическом, пластиковом и картонном ящиках хранится неисчерпаемый запас разнообразныъ фруктов: яблок, апельсинов, бананов, груш и киви.

За один раз можно достать лишь один фрукт из одного ящика не глядя, при этом требуется соблюдать следующие правила:
- Из металлического ящика можно доставать не больше 30 фруктов в минуту
- Из деревянного ящика можно доставать не больше 50 фруктов в минуту
- Из пластикового ящика можно доставать не больше 40 фруктов в минуту
- Из картонного ящика можно доставать не больше 65 фруктов в минуту

Фрукты надо переместить в бездонные пакеты: красный, синий, зелёный, жёлтый и оранжевый, — соблюдая следующий правила сортировки:
- Красный пакет не принимает яблоки
- Синий пакет не принимает никакие фрукты из пластикового ящика
- Зелёный пакет принимает не больше 10 яблок в минуту
- Жёлтый пакет не принимает киви из пластикового ящика
- Оранжевый пакет принимает любые фрукты из металлического ящика и только бананы из прочих ящиков

По возможности следует равномерно распределять фрукты по пакетам, подходящим под условия сортировки.

Фрукт, вынутый из ящика и не подходящий ни под одно из условий сортировки, выбрасывается.

Раунд сортировки длится 5 минут.

На экран выводится отчет: сколько фруктов каждого вида в каждом пакете, а также сколько фруктов было выброшено.
